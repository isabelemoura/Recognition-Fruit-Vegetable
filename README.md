# Reconhecimento de Frutas e Vegetais

Este é um projeto de reconhecimento de alimentos utilizando a biblioteca TensorFlow para aprendizado de máquina e visão computacional. O objetivo é construir um modelo que seja capaz de identificar diferentes tipos de frutas e vegetais a partir de imagens.

## Instalações necessárias

Antes de executar o projeto, é necessário instalar algumas bibliotecas Python. Você pode fazer isso facilmente utilizando o comando pip. Certifique-se de ter o Python instalado em seu ambiente. Caso ainda não tenha, você pode baixá-lo em [https://www.python.org/](https://www.python.org/).

Para instalar as bibliotecas necessárias, abra um terminal ou prompt de comando e digite os seguintes comandos:

```bash
pip install tensorflow
pip install matplotlib
pip install wget
pip install kaggle
```

## Kaggle Dataset
O conjunto de dados utilizado neste projeto foi obtido do Kaggle e está disponível no seguinte link: Kaggle - Fruit and Vegetable Image Recognition

O dataset contém uma grande variedade de imagens de frutas e vegetais, rotuladas com as categorias corretas. Essas imagens serão utilizadas para treinar o modelo de reconhecimento de alimentos.

Espero que este projeto de reconhecimento de alimentos seja útil para diversos propósitos, desde aplicativos de smartphones até soluções para a indústria alimentícia.
